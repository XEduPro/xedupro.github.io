XEduPro是一套开箱即用的人工智能学习工具，集成多个功能库，帮助您快速掌握AI技术。

一键安装包

包含MMEdu、BaseML、BaseNN三个模块的基础功能，以及XEduHub、BaseDT、BaseDeploy等工具库，同时内置了一套EasyDL系列工具。解压即可使用，适合初学者。

系统要求

最低配置
  • CPU: 双核 2.0 GHz
  • 内存: 4GB RAM
  • 存储: 5GB 可用空间
  • 操作系统: Windows 10+, macOS 10.15+, Ubuntu 18.04+
推荐配置
  • CPU: 四核 3.0 GHz
  • 内存: 8GB RAM
  • 显卡: NVIDIA GPU (可选，用于加速)
  • 存储: 20GB 可用空间
高级配置
  • CPU: 六核以上 3.5 GHz
  • 内存: 16GB+ RAM
  • 显卡: NVIDIA RTX系列
  • 存储: SSD 50GB+ 可用空间
